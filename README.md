# Installation

Ubuntu Linux

1. $ sudo apt-get install virtualbox vagrant
2. $ sudo bash
3. $ echo "192.168.88.88 dev.vision.local" >> /etc/hosts
4. $ exit
5. $ cd yourprojectsdir
6. $ mkdir vision && cd vision
7. $ git clone https://github.com/SnowBeaver/Vision.git project
8. $ cd project
9. $ vagrant up
